Scroll Reveal Animations

Overview:

This demo uses a few of the animations from animate.css to create scroll animations. However, other animate.css can be easily brought in and used. Note: Some of the animations have also been modified from what they originally were.

Slide In Left:

Gotcha: In order for the slide in and out animations to go completely off screen, the animated section has to take up 100% of the viewport width. Often this isn't the case with content because it is inside of other containers that have horizontal padding and/or margin. Thus, one either has to remove any intervening horizontal padding and/or margins, or tweak the animation itself.

Flip In X:

Gotcha: Because the the section is actually rotated it won't trigger the animation at exactly the same spot. Rather you will have to scroll down 50% of the section's height before it does anything. Consequently, if you wer going to use this animation, you might need a more elaborate scroll handler.

Flip In Y:

Gotcha: The problem with this animation is that when the section is rotated it is actually much higher in the scroll than we want it to be. This breaks the effect when the user is scrolling too slow. It can be mitigated by using a very high perspective - e.g., perspective(3000px)) - but the problem will never be entirely fixed.

Consequently, y-axis animations seem to be bad candidates for scroll animation, unless one uses a more elaborate handler.
